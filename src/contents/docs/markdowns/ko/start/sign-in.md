# Sign in

작업을 저장하고 공유하려면 계정이 필요합니다. Simulizer는 Google 계정으로
로그인하므로, 따로 만들어 둘 비밀번호가 없습니다.

## Steps

1. 홈 화면 오른쪽 위 **무료로 시작**(또는 **로그인**) 을 누르세요.
2. Google 로그인 창이 열립니다. Gmail 계정으로 로그인하세요.
3. 권한 동의 후 자동으로 **대시보드**로 이동합니다.

대시보드는 파일 목록을 보여 줍니다. 파일 하나가 곧 작업 하나입니다.
새 계정은 아직 비어 있으니, 다음 페이지에서 첫 작업을 만들어 봅니다.

## Notes

- 파일은 기본적으로 **비공개**입니다. 본인만 열 수 있습니다.
- 공유 여부는 작업마다 직접 정합니다. [공유와 복제](/docs/tools/share-and-duplicate)를 참고하세요.
- 계정을 삭제해도 30일 동안은 복구할 수 있습니다.
