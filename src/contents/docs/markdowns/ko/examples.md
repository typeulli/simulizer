# Examples

다른 사람들이 Simulizer로 만든 작업을 열어 보세요. 각 카드는 작업을
*읽기 전용*으로 열며, 자기 사본을 만들고 싶다면 워크스페이스에서
**복제**를 누르면 됩니다.

> 이 갤러리는 아직 채워지는 중입니다. 새 예시는 앞으로 계속
> 추가됩니다.

## Getting started

### Sine curve

가로축을 200개의 점으로 나눠 `sin(x)`를 그립니다.
[첫 결과](/docs/start/first-result) 페이지에서 만든 작업의 완성본입니다.

- **여는 링크** — `/workspace?file=GwurXOiZ`
- **카테고리** — 수학 · 함수
- **블록 수** — 약 12
- **실행 시간** — 1초 미만

## How to add a new example

작업을 [공유 링크](/docs/tools/share-and-duplicate)로 바꾼 뒤, 위
형식과 똑같이 이 페이지에 카드를 추가하세요. 각 카드는 다음 네 줄이면
충분합니다:

- 한 줄 제목
- 한 단락 설명 (무엇을 보여 주는지, 어떤 블록을 쓰는지)
- 워크스페이스 링크 (`/workspace?file=<공유ID>`)
- 분류 · 블록 수 · 실행 시간 같은 메타
